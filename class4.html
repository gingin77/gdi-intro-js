Events

An 'event' is a type of object that is created when the user interacts with a web page.

For example, an event is emitted when a user clicks an element. That event is named "click"

Event handlers

You can add event handlers (also called event listeners) to elements.

Event handlers are functions that get called when the event happens.


function fireMissiles(event) {
  // code to execute when the click happens
};

// first way to add an event handler
element.onclick = fireMissiles;

// second way to add event handler(s)
element.addEventListener('click', fireMissiles);

Types of Events

There are a tons of events in javascript. Some of the most common:

  • click
  • mouseover
  • keyup
  • load
  • focus

Events and Code

How do you run some code when an event is emitted?

  1. Create a function with the code you want to run
  2. Add an event listener and provide your function as the handler

Calling Functions from HTML

As a shortcut, you can use the onclick attribute directly on the HTML element:


function sayHi(){
  alert('Hi!');
}



<button id="clickMe" onclick="sayHi()">Click Me!</button>

Default Behavior

When an event handler (a function) is called, it receives the event object as the first argument.

Certain events (like clicking links) have default behaviors. The event object has a built-in method to prevent that behavior:


var linkElement = document.getElementById('fakeLink');

linkElement.onclick = function (event) {
  // instead of navigating to the page that the link points to,
  // this will do nothing
  event.preventDefault();
};

this in event handlers

Inside of an event handler, the this keyword (context) will hold a reference to the element that the event emitted from:



var clickElem = document.getElementById('click-element');

clickElem.onclick = function removeElement(event){
  this.remove();
}

Forms

A form is an excellent way to collect data from a user:


<form id="temperatureForm">
  <label for="temp">Temperature:</label> <input type="text" id="temp" size="3"/> degrees in
  <input type="radio" name="tempFormat" value="F" checked />Fahrenheit
  <input type="radio" name="tempFormat" value="C" />Celsius <br />
  <label for="submitButton"></label> <input id="tempSubmitButton" type="submit" value="Submit" />
</form>

Retrieving Form Data

You retrieve the values of form elements using the value property:



tempForm.onsubmit = function(event) {
  event.preventDefault();

  var temperature = document.getElementById('temp').value;
  // also try this.elements!

  console.log(temperature);
}

Radio Buttons

Radio buttons usually do not have IDs, so you will need find the one that is selected another way:



var scale =
  document.querySelector('input[name="tempFormat"]:checked').value;

The Submit Event

If you want to listen for a form submit event, be sure to prevent the default behavior!



var tempForm = document.getElementById('tempForm');

function logTemperature(event){
  event.preventDefault();
  var temperature = document.getElementById('temp').value;
  console.log(temperature);
}

tempForm.addEventListener('submit', logTemperature, false);

Timing Functions

In JavaScript, it is possible to execute some code at specified time intervals.

setInterval() executes a given function over and over again, at a specified time interval
setTimeout() executes a given function, once, after waiting a specified number of milliseconds

setInterval

The function signature for setInterval is:


setInterval(function, milliseconds);

For example, this is a simple clock:


var intervalId = setInterval(function() {
    var currentTime = new Date();
    var timeString = currentTime.toLocaleTimeString();
    document.body.innerHTML = timeString;
}, 1000);

clearInterval

You can use clearInterval() to stop a setInterval loop:


clearInterval(intervalId);

setTimeout

The method setTimeout() is similar, but it will only run once:


setTimeout(function, milliseconds);

setTimeout

For example, this code will wait 3 seconds, then create a popup box, unless the user triggers the clearTimeout():


var timeoutId = setTimeout(function() {
  alert('Hello');
}, 3000);

document.body.onclick = function cancelHello(event) {
  clearTimeout(timeoutId);
}

Animations

By changing values slowly over time, we can create simple animations:


var targetPic = document.getElementById('turtlePic');

targetPic.onclick = function () {
  var leftMarginValue = 0;
  // increase every 10ms
  var intervalId = setInterval(function moveImage() {
    // increase margin every time
    leftMarginValue++;
    // set new left margin
    targetPic.style.marginLeft = leftMarginValue + 'px'
    // are we done?
    if (leftMarginValue === 200) {
      clearInterval(intervalId);
    }
  }, 10);
}
