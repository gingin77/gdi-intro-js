Scope

Scope is the concept of what's visible and invisible at certain parts of the programâ€”what the program can see and has access to (variables, functions). It's all about visibility.

Global Scope

Things that are visible everywhere in the program are said to live in the "Global Scope"

Local Scope

A variable declared inside of a function has a local scope and can only be seen within that function.

Boolean Variables

Boolean variables represent the logical values True and False


		var customerHasPaid = true;
		var showVideo = false;

If you try to use another variable as a boolean, JavaScript will guess. The number 0, the empty string ' ', undefined, and null are considered false, everything else is "truthy".

Comparison Operators

Use this page as a reference for equal vs identical
Example Name Result
a == b Equal TRUE if a is equal to b (can be different types).
a === b Identical TRUE if a is equal to b, and the same type.
a != b Not equal TRUE if a is not equal to b (can be different types).
a !== b Not identical TRUE if a is not equal to b, or they are not the same type.
a < b Less than TRUE if a is strictly less than b.
a > b Greater than TRUE if a is strictly greater than b.
a <= b Less than or equal to TRUE if a is less than or equal to b.
a >= b Greater than or equal to TRUE if a is greater than or equal to b.

Watch out!

Don't mix up = and ==

A single equal sign serves as an assignment operator, ie for defining variables

A double equal sign is used to compare if 2 things are equivalent

A helpful way to distinguish the two might be thinking of single "=" as a making a statement vs the double "==" as asking a question.

While loops

The while loop will repeat the same code over and over until a predefined condition is satisfied.


var beersLeft = 99;

while (beersLeft > 1) {
	alert(beersLeft + ' bottles of beer on the wall');
	beersLeft -= 1;
}

Infinite Loops

Make sure something changes in the loop, or your loop will go on forever...

For loops

For loops are very similar, but you declare a counter in the statement.


		//will count 1 to 10
		for (var i = 1; i <= 10; i++) {
		console.log(i);
		}

Loops and logic

You can add other statements or logical operators inside the loops.


//Count from 1 to 50
for (var i = 1; i <= 50; i++) {
	console.log(i);

	// say "fizz" if i is a multiple of three
	if (i % 3 === 0) {
	console.log(' fizz');
	}

	// say "buzz" if i is a multiple of three
	if (i % 5 === 0) {
	console.log(' buzz');
	}

	console.log( "FizzBuzz" );
}

Break

To exit a loop, use the break statement.

										
// count from 100 to 200
for (var i = 100; i <= 200; i++) {

	console.log('good so far: ' + i);

	// stop at the first multiple of 7
	if (i % 7 == 0) {
		console.log('Found a multiple of 7! - ' + i);
		break;
	}
}

Array Length

The length property tells you how many items are in an array


var tags = ['#javascript', '#hacker', '#gdirdu'];
console.log(tags.length); // 3

Using Arrays

Arrays are zero-indexed, meaning the first position is known as position 0. The last position will always be the array length minus one.

You can access items with "bracket notation" by using the position of the item you want.


var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet'];
var firstColor = rainbowColors[0];
var lastColor = rainbowColors[6];

console.log(firstColor, lastColor);

Changing arrays

You can use bracket notation to change an item in an array



var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

function logFirstFavorite() {
	console.log(myFavoriteThings[0]);
}

logFirstFavorite();

myFavoriteThings[0] = 'Garlic Roasted Brussel Sprouts';

logFirstFavorite(); // 'Garlic Roasted Brussel Sprouts'

Expanding arrays

Arrays do not have a fixed length. You can use "push" to add something to an array.


var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];
myFavoriteThings.push('Dancing');

console.log(myFavoriteThings);

Iterating through arrays

Use a for loop to easily do something with each item in an array.


var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet'];

// loop as many times as items we have
for (var i = 0; i < rainbowColors.length; i++) {

	// each time this runs, i will be 1 number higher and
	// effectively "walk" up our array

	console.log(rainbowColors[i]);
}

Objects

Objects let us store a collection of properties.


var objectName = {
  propertyName: propertyValue,
  propertyName2: propertyValue,
  ...
};
			  

var aboutMe = {
	hair: "brown",
  eye_color: 'blue',
	likes: ['dogs', 'coding', 'mint chocolate chip ice cream']
};

Accessing Objects

You can retrieve values using "dot notation"


var aboutMe = {
	hair: "brown",
  eye_color: 'blue',
  likes: ['dogs', 'coding', 'mint chocolate chip ice cream']
};
var myEyeColor = aboutMe.eye_color;

alert(myEyeColor);

Or by using "bracket notation" (like arrays)


	var myHairColor = aboutMe["hair"]

Or by combining dot notation and bracket notation together to access an item in an array in an object


var anotherFavoriteThing = aboutMe.likes[2]
var aFavoriteThing = aboutMe["likes"][1]

Changing Objects

You can use dot or bracket notation to change properties


var aboutMe = {
  hometown: 'Charlotte, NC',
  hair: 'blonde'
};

aboutMe.hometown = 'Durham, NC';
aboutMe['hair'] = 'brown';

console.log(aboutMe.hometown); // Durham, NC
console.log(aboutMe.hair); // brown

A website is a way to present your content to the world, using HTML and CSS to present that content & make it look good.

Refresher: IDs vs. Classes

ID -- Should only apply to one element on a webpage. For instance, a webpage only has one footer.

The "#" is how you tell CSS "this is an id."

Class -- Lots of elements can have the same class. For instance, there can be many warnings on one webpage.

The "." is how you tell CSS "this is a class name."

The DOM Tree

We model the nested structure of an HTML page with the DOM (Document Object Model) Tree. The browser makes a "map" of all the elements on a page.

The DOM: Sample Code


<!DOCTYPE html>
<html>
    <head>
        <title>Test Page</title>
        <style>
            h1 {
             color: red;
            }
        </style>
    </head>
    <body>
        <h1>My Page</h1>
        <p>Hello World!</p>
        <img src="http://placebear.com/200/300" alt="bear"/>
    </body>
</html>

The DOM: Sample Model

DOM Access

Your browser automatically builds a Document Object to store the DOM of a page. To change a page:

  1. Find the DOM node and store it in a variable
  2. Use methods to manipulate the node

DOM Access: By Id

You can find nodes by id using the method:


document.getElementById(id);

To find:


<img id="bearPic" src="http://placebear.com/200/300" alt="bear"/>

We would use:


var imgKitten = document.getElementById('bearPic');

DOM Access: By Tag Name

You can find certain types of HTML elements using this method:


document.getElementsByTagName(tagName);

To find:


<li>Daisy</li>
<li>Tulip</li>

We would use:


var listItems = document.getElementsByTagName('li');
for (var i = 0; i < listItems.length; i++) {
  var listItem = listItems[i];
}

DOM Access: HTML 5

In newer browsers, you can use other methods too.

Available in IE9+, FF3.6+, Chrome 17+, Safari 5+:


document.getElementsByClassName(className);

Available in IE8+, FF3.6+, Chrome 17+, Safari 5+:


document.querySelector(cssQuery);
document.querySelectorAll(cssQuery);

getElement vs. getElements

Any method that starts with "getElement" will return a single node.


document.getElementById('uniqueID'); //returns a single node

Any method that starts with "getElements" will return a array of nodes. To modify a single node, you will need to use bracket notation to select the correct one.


document.getElementsByTagName('p'); //returns multiple nodes
var specficParagraph = document.getElementsByTagName('p')[2];

DOM Nodes: Attributes

You can access and change attributes of DOM nodes using dot notation.

To change this element:


<img id="bearPic" src="http://placebear.com/500/500" alt="cat"/>

We could change the src attribute this way:


var imgBear = document.getElementById('bearPic');
imgBear.src = 'https://placebear.com/g/200/300';

DOM Nodes: Getting and Setting Attributes

You can also use getAttribute/setAttribute:


<img id="bearPic" src="http://placebear.com/500/500" alt="bear"/>

We could change the src attribute this way:


var imgBear = document.getElementById('bearPic');
imgBear.setAttribute('src', 'https://placebear.com/g/200/300');

DOM Nodes: Styles

You can change page css using style

To make this CSS:


body {
  color: red;
}

Use this JavaScript:


var pageNode = document.getElementsByTagName('body')[0];
pageNode.style.color = 'red';

DOM Nodes: More Styles

When using style, change any CSS property with a "-" to camelCase, and be sure to include a unit on any number

To make this CSS:


body {
  background-color: pink;
  padding-top: 10px;
}

Use this JavaScript:


var pageNode = document.getElementsByTagName('body')[0];
pageNode.style.backgroundColor = 'pink';
pageNode.style.paddingTop = '10px';

DOM innerHTML

Each DOM node has an innerHTML property with the HTML of all its children. You can use the innerHTML property to view or change the HTML of a node.

For example, you can overwrite the entire body:


var pageNode = document.getElementsByTagName('body')[0];
pageNode.innerHTML = '<h1>Oh Noes!</h1> <p>I just changed the whole page!</p>'

Or just add some new content to the end:


var pageNode = document.getElementsByTagName('body')[0];
pageNode.innerHTML += '...just adding this bit at the end of the page.';

DOM innerHTML continued

You can also target a particular element

To fill this HTML element:


<p id="warning"></p>

We can select the node and modify it:


var warningParagraph = document.getElementById('warning');
warningParagraph.innerHTML = 'Danger Will Robinson!';

Creating New Nodes

The document object also provides ways to create nodes from scratch:


document.createElement(tagName);
document.createTextNode(text);
document.appendChild();

Creating New Nodes: Sample Code


var pageNode = document.getElementsByTagName('body')[0];
var newImg = document.createElement('img');

newImg.src = 'http://placebear.com/400/300';
newImg.style.border = '1px solid black';
pageNode.appendChild(newImg);

var newParagraph = document.createElement('p');
var paragraphText = document.createTextNode('Squee!');

newParagraph.appendChild(paragraphText);
pageNode.appendChild(newParagraph);
